# Organizador Web de Tareas - Fase 2

Proyecto final de Aplicaciones y Programación Web.

## Descripción
Aplicación web sencilla para registrar, organizar, filtrar, completar y eliminar tareas personales.
Los datos se guardan en el navegador mediante localStorage.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- localStorage
- GitHub Pages

## Archivos principales
- index.html: estructura principal del sitio.
- css/estilos.css: estilos visuales y responsive design.
- js/app.js: validaciones, eventos, DOM y localStorage.

## Cómo publicar en GitHub Pages
1. Crear un repositorio público en GitHub.
2. Subir index.html, css, js, img y .nojekyll.
3. Entrar en Settings > Pages.
4. En Build and deployment, elegir Deploy from a branch.
5. Seleccionar branch main y carpeta /root.
6. Guardar y esperar la URL publicada.
